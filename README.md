# Playwright 기반 서비스 헬스 체크 자동화 (PoC)


본 프로젝트는 주요 포털 사이트(Daum, Naver)의 배포 직후 서비스 안정성을 신속하게 검증하기 위해 구축된 하이브리드 자동화 도구입니다.

## 1. 개요
기존의 단순 수동 점검 방식에서 벗어나, API 응답과 UI 링크 연결 상태를 동시에 검증하여 QA 공수를 절감하고 배포 품질을 확보하는 것을 목적으로 합니다.

## 2. 주요 기능
- **API Health Check**: 메인 서버의 HTTP 응답 상태 코드를 확인하여 서버 생존 여부 즉각 판별
- **UI Broken Link 전수 조사**: 페이지 내 모든 `<a>` 태그를 수집하여 링크 깨짐(404, 500 등) 여부 자동 전수 조사
- **보안 시스템(WAF) 회피**: 
  - 일반 브라우저와 동일한 `User-Agent` 헤더 적용
  - 사람의 행동 패턴을 모방한 랜덤 딜레이(0.5s ~ 1.5s) 로직 구현
- **자동 리포팅**: 테스트 결과를 `CSV` 파일로 자동 생성하여 점검 이력 관리 및 결함 보고 용이

## 3. 기술 스택
- **Language**: TypeScript
- **Framework**: Playwright
- **Runtime**: Node.js
